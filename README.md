## About The Project

A clean, beautiful, responsive and 100% customizable portfolio template for Developers! This project is build using React.js. It is a SPA(Single Page Application) that uses react-router-dom to navigate between pages. It is fully responsive and uses bootstrap to make it responsive. It is also a PWA(Progressive Web App) that can be installed on any device.

## Getting Started

This section will guide you to run this project on your local machine.

### Prerequisites

You need to have `Node.js` and `npm` installed on your machine. If you don't have them installed, you can install them from [here](https://nodejs.org/en/download/).

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/ashutosh1919/masterPortfolio.git
   ```
2. Move to the project directory
   ```sh
   cd masterPortfolio
   ```
3. Install NPM packages
   ```sh
   npm install
   ```
4. Run the project
   `sh npm start`
   This will run the project on your local machine. You can view it on `http://localhost:3000/`.

## How To Use

This project is a template for your portfolio. You can customize it to your needs. The following sections will guide you to customize this project.

### `src/portfolio.js`

This file contains all the information about you. You need to change the information in this file to customize the portfolio.
The `portfolio.js` file contains the following sections:

- `greeting`: This section contains the information about the home page.
- `socialMediaLinks`: This section contains the links to your social media profiles.
- `skills`: This section contains the information about your skills.
- `degrees`: This section contains the information about your education.
- `certifications`: This section contains the information about your certifications.
- `experience`: This section contains the information about your experience.
- `projects`: This section contains the information about your projects.

### `src/theme.js`

This file contains the theme of the portfolio. You can change the theme of the portfolio by changing the colors in this file.
